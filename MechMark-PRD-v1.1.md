# MechMark 產品需求文件 (PRD) v1.1

> 給機構工程師專用的螢幕截圖標註工具。
> 核心差異化：把「畫一個工程尺寸標註」從拼四個零件簡化成一次手勢。

---

## 0. 一句話定位

機構工程師截圖後，要快速畫出**像工程圖一樣的尺寸標註、半徑/直徑、箭頭、圈選**，
再複製或存檔貼進報告/email/票務系統。市面工具的標註器都是「畫線玩具」，
沒有一個把尺寸標註的工程語意做對 —— MechMark 補的就是這個洞。

---

## 1. 目標使用者與情境

- **使用者**：機構/結構工程師（CREO/SolidWorks 使用者），日常在 Windows 上工作。
- **典型截圖來源**：CAD 圖面、3D 模型視窗、PDF 工程圖。
- **典型動作**：截一段 → 標幾個尺寸/箭頭/圈選 → 數字自己從 CAD 讀後手打 → 複製或存檔。
- **頻率**：一天數十次，所以「每個標註少點兩下」就是真價值。

---

## 2. 目標 / 非目標（範圍紀律）

### 2.1 v1 目標
- 系統熱鍵擷取螢幕區域 → 進入標註畫布。
- 一套「工程語意正確」的標尺引擎（線性、角度、半徑、直徑）。
- 折線引線註解（箭頭 → 折線 → 水平肩線 → 文字）。
- 箭頭、矩形、圓/橢圓、文字等標記工具。
- 統一的「當前樣式」系統（線粗、文字大小、箭頭大小、顏色、實線/虛線）。
- 複製到剪貼簿 + 存成 PNG。

### 2.2 明確非目標（v1 不做，避免範圍爆炸）
- **不做真實單位量測/比例校準**：數字一律手動輸入，工具不換算 px→mm。
- **不讀 CAD 檔**（DWG/STEP）：只處理點陣截圖。
- **不做影像幾何吸附**：不對截圖內容做邊緣偵測自動 snap（Phase 2 再評估）。
- **不做雲端/帳號/分享**：純本機。
- **不做 OCR、不直接標 PDF**：要標 PDF 先截圖。
- **不支援 Mac / 行動裝置**：v1 只做 Windows。

## 3. 平台與技術選型

| 項目 | 決定 | 理由 |
|---|---|---|
| OS | Windows 10/11 | 使用者工作環境 |
| 殼層 | **Electron** | 全 JS 生態，全域熱鍵與螢幕擷取有成熟範例，vibe coding 最不易卡死；Tauri 後端 Rust 是非程式背景者的撞牆點 |
| 標註引擎 | HTML5 Canvas | 原型已驗證可行，可幾乎原封不動移植 |
| 擷取 | `globalShortcut` + `desktopCapturer`（或原生模組） | 標準作法 |
| 匯出 | `clipboard.writeImage` + 存檔 | 對應使用者需求 |

> **已知風險（必讀）**：擷取模組是整個專案最花 debug 時間的地方，
> 多螢幕與 Windows DPI 縮放幾乎一定會出問題。開發順序故意把這段排在標尺引擎**之後**，
> 先把有價值且好做的核心做穩，再處理痛苦的外殼。

## 4. 核心流程

```
[ 按熱鍵 ] → [ 全螢幕凍結+變暗遮罩 ] → [ 拖曳框選區域 ]
        → [ 裁切影像載入標註畫布 ] → [ 標註 ] → [ 複製 / 存 PNG ]
```

## 5. 「當前樣式」系統（FR-STYLE）

所有可視屬性收斂成一組當前樣式，新物件繼承之，選取舊物件可單獨覆寫。

| 屬性 | 範圍 / 選項 | 預設 |
|---|---|---|
| 線粗 lineWidth | 1–8 px | 標記類 2.5；尺寸線 1.4 |
| 文字大小 textSize | 10–28 px | 15 |
| 箭頭大小 arrowSize | 小/中/大 | 中 |
| 顏色 color | 紅/藍/綠/黑/白 + 自訂 | 紅 #E5342B |
| 線型 lineStyle | 實線 / 虛線 | 實線 |

- FR-STYLE-1：樣式列即時反映當前值；改值套用到「接下來要畫的」物件。
- FR-STYLE-2：選取既有物件改樣式 → 即時覆寫該物件。
- FR-STYLE-3：框選多個物件可一次套用樣式。
- FR-STYLE-4：尺寸線預設較細，獨立於標記類線粗。

## 7. 線性標尺引擎（FR-DIM）— 核心

### 7.1 操作狀態機
| 狀態 | 動作 | 說明 |
|---|---|---|
| S1 | 點起點 A | — |
| S2 | 點終點 B | 正交鎖定 ON 時自動吸附成純水平或垂直（取 ΔX/ΔY 較大方向）；按住 Shift 臨時反轉鎖定狀態 |
| S3 | 移動滑鼠定位 | 即時預覽完整尺寸；尺寸線偏移 = 滑鼠到 A–B 線的垂直帶號距離 |
| S4 | 點一下鎖定 → 跳輸入框 | 打數字，Enter 完成 |

- FR-DIM-1：完成一個帶數字的尺寸，總操作 = 三次點擊 + 輸入。**這是驗收紅線。**
- FR-DIM-2：**正交鎖定預設 ON**，提供常駐切換鈕關閉；Shift 為臨時反轉鍵。
- FR-DIM-3：S1–S3 任一階段按 Esc 取消整個標註；S4 按 Esc 以空值完成。

### 7.2 幾何與外觀（工程圖語意）
- FR-DIM-4：水平/垂直鎖定時，**兩條延伸線長度依各自量測點自動計算**，
  量測點不等高時延伸線長度本來就不同（正確 CAD 行為）。
- FR-DIM-5：**每條延伸線在選取模式下有獨立端點手把，可手動拖曳調長短**；
  手動值覆寫自動值，可重置回自動。
- FR-DIM-6：延伸線起點離量測點留小間隙（gap ≈ 4px），末端略超出尺寸線（overshoot ≈ 7px）。
- FR-DIM-7：尺寸線兩端箭頭，外觀為細長 ISO 風（非馬克筆粗箭頭）。
- FR-DIM-8：數字壓白底、隨尺寸線方向旋轉並保持正立可讀。
- FR-DIM-9：數字為自由文字，支援 `Ø R ± ° ⌀` 與公差寫法（如 `12.5±0.1`）。
- FR-DIM-10：提供 `Ø R ± ° ⌀` 快捷插入鈕（一般鍵盤難打，每天要用）。

### 7.3 小尺寸自動降級（FR-DIM-SMALL）— 標準 CAD 兩段式
令可用內距 `S` = 兩延伸線之間沿尺寸線的距離；
`need = 數字寬度 + 2×箭頭長 + padding`。

| 階段 | 條件 | 行為 |
|---|---|---|
| 正常 | S ≥ need | 箭頭在內側指向兩端延伸線，數字置中 |
| 降級一 | 數字寬 ≤ S < need | **箭頭翻到延伸線外側**（朝內指），尺寸線略延伸，數字仍置中 |
| 降級二 | S < 數字寬 | 箭頭維持外翻 + **數字以折引線拉到外側**（短肩線），引線指回尺寸線中點 |

- FR-DIM-SMALL-1：階段判定即時計算，標註後改線粗/文字大小會自動重算階段。
- FR-DIM-SMALL-2：降級二的數字外移方向預設取空間較大的一側，完成後可手動拖曳數字位置。

### 7.4 編輯
- FR-DIM-EDIT-1：拖端點重新量測；拖中段平移整個尺寸偏移。
- FR-DIM-EDIT-2：雙擊數字可改值。

## 8. 半徑 / 直徑標尺（FR-CIRC）— 三點抓圓

### 8.1 共同：三點外接圓擬合
- FR-CIRC-1：依序點圓周上三點 → 解兩條垂直平分線交點得圓心 C 與半徑 r（外接圓）。
- FR-CIRC-2：三點近共線（退化）時拒絕並提示重點。
- FR-CIRC-3：擬合過程淡淡顯示推得的圓供確認，完成後隱藏。
- FR-CIRC-4：三個擬合點皆可在選取模式拖曳，重新擬合圓。

### 8.2 半徑（R）
- 擬合後移動滑鼠決定引線角度（指向游標方向的圓弧點），點一下鎖定。
- 圓心 → 圓弧單箭頭線；數字自動加 `R` 前綴，可手改。

### 8.3 直徑（Ø）
- 擬合後移動滑鼠決定直徑線角度，點一下鎖定。
- 穿過圓心、兩端到圓弧的雙箭頭線；數字自動加 `Ø` 前綴，可手改。

> 小直徑/半徑同樣套用 §7.3 的降級規則。

## 9. 標記工具（FR-MARK）

- FR-MARK-1 箭頭：拖曳畫出；選取後可**反向**（換頭尾，熱鍵 R 或拖端點）。
- FR-MARK-2 矩形：拖曳畫出；可切實線/虛線。
- FR-MARK-3 圓/橢圓：拖曳外接框畫出，按住 Shift 畫正圓；可切實線/虛線。
- FR-MARK-4 文字：點擊放置，自由輸入；文字大小吃當前樣式。

## 9.5 角度標尺（FR-ANG）— 兩線四點式

### 9.5.1 操作狀態機
| 狀態 | 動作 | 說明 |
|---|---|---|
| S1 | 點第一條線起點 P1 | — |
| S2 | 點第一條線終點 P2 | 定義向量 V1 = P2–P1 |
| S3 | 點第二條線起點 P3 | — |
| S4 | 點第二條線終點 P4 | 定義向量 V2 = P4–P3；解出兩線交點 O（實際或延長線上） |
| S5 | 移動滑鼠定位弧線半徑 | 即時預覽角度弧與數字；半徑 = 游標到 O 的距離 |
| S6 | 點一下鎖定 → 跳輸入框 | 打角度值（預填計算值），Enter 完成 |

- FR-ANG-1：S1–S5 任一階段按 Esc 取消。
- FR-ANG-2：兩線平行（無交點）時拒絕並提示重點。
- FR-ANG-3：交點 O 落在量測點範圍外時（延長線交點），延伸虛線輔助線顯示交點位置，完成後隱藏。

### 9.5.2 幾何與外觀
- FR-ANG-4：弧線連接兩向量方向上距 O 等半徑處，兩端各一個箭頭（ISO 風格）；當弧太短塞不下兩個箭頭時，箭頭移至弧外側（同線性降級邏輯）。
- FR-ANG-5：顯示的角度為兩向量夾角（0°–180°）；游標在補角一側時自動切換到補角顯示，讓使用者自然選想標的那個角。
- FR-ANG-6：數字加 `°` 後綴，壓白底，置於弧中點外側；文字大小吃當前樣式。
- FR-ANG-7：輸入框預填計算角度（小數點後一位），使用者可直接 Enter 接受或手動覆寫。
- FR-ANG-8：選取後可拖曳 P1–P4 四個端點重新定義兩線；可拖曳數字位置。
- FR-ANG-9：雙擊數字可改值。

## 9.6 折線引線註解（FR-LEADER）— 標準 CAD callout

### 9.6.1 操作狀態機
| 狀態 | 動作 | 說明 |
|---|---|---|
| S1 | 點箭頭端（指向目標） | — |
| S2 | 點折點（肘點） | 定義斜線段方向；可拖曳肩線長度預覽 |
| S3 | 移動滑鼠決定肩線方向與長度 | 自動鎖水平（肩線永遠水平）；預覽文字掛在肩線端 |
| S4 | 點一下鎖定 → 跳輸入框 | 打文字，Enter 完成；支援多行（Shift+Enter 換行） |

- FR-LEADER-1：S1–S3 任一階段按 Esc 取消。
- FR-LEADER-2：**肩線永遠水平**，長度由 S3 滑鼠位置決定（游標在折點左側 → 肩線向左；右側 → 向右）；完成後可拖曳肩線端點改長度與方向。
- FR-LEADER-3：文字掛在肩線端，靠肩線方向對齊（右出 → 左對齊；左出 → 右對齊）。

### 9.6.2 幾何與外觀
- FR-LEADER-4：箭頭端為單箭頭（ISO 風格），指向 S1 點；斜線段為實線；肩線為實線。
- FR-LEADER-5：文字大小吃當前樣式；支援 `Ø R ± °` 等特殊字元（同 §7.2 FR-DIM-10 快捷鈕）。
- FR-LEADER-6：選取後可拖曳箭頭點（S1）、折點（S2）、肩線端點三個手把獨立調整；雙擊文字可改內容。
- FR-LEADER-7：折點到箭頭的斜線與水平夾角建議維持 30°–60° 為佳，但不強制限制（使用者自行判斷）。

## 10. 匯入 / 匯出（FR-IO）

- FR-IO-1 擷取：系統熱鍵 → 全螢幕遮罩 → 框選 → 載入畫布。
- FR-IO-2 旁路擷取：也支援 Ctrl+V 貼上、拖曳圖檔、開檔（給臨時用/相容用）。
- FR-IO-3 複製：一鍵複製含標註的影像到剪貼簿。
- FR-IO-4 存檔：存成 PNG。
- FR-IO-5 多螢幕 / DPI：擷取需正確處理多顯示器與 Windows 縮放（已知高風險，列為擷取模組驗收重點）。

## 12. 驗收標準（差異化功能，必過）

1. 從選工具到完成一個帶數字的水平尺寸標註 ≤ 三次點擊 + 輸入。
2. 正交鎖定預設開啟；兩量測點不等高時延伸線自動不等長。
3. 每條延伸線可手動拖曳調整長短，並可重置回自動。
4. 小尺寸依「箭頭外翻 → 數字外移引線」兩段式自動降級，且改樣式後即時重算。
5. 三點可抓出圓並正確產生 R / Ø 標註，三點可拖曳重新擬合。
6. 矩形與圓可切換實線/虛線；箭頭可反向。
7. 標註後可複製到剪貼簿並貼進 Word/PPT；可存 PNG。
8. 角度標尺：點四點定義兩線 → 移動決定弧半徑 → 輸入角度，共六個動作完成；游標跨補角自動切換顯示角度。
9. 引線註解：箭頭點 → 折點 → 肩線自動水平 → 輸入文字，四個動作完成；三個手把可獨立拖曳。

## 14. 開發順序建議

1. **標尺引擎**（§5–§9）做成可獨立執行的網頁，反覆打磨手勢與外觀 —— 價值最高、最好做。
2. 確認核心好用後，**包進 Electron**，把畫布原封不動搬入。
3. 最後接**擷取模組**（§10），單獨對付多螢幕/DPI。
4. 接上**複製/存檔**，收尾。
